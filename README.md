# Bookstore Tableau Dashboard

## Project Overview

This Tableau project analyses bookstore sales performance across genres, regions, publishers and order sizes.

The project uses Tableau visualisations to explore Q1 sales trends, regional performance, genre performance, publisher contribution and the relationship between sales and quantity.

## Tools Used

- Tableau
- Data visualisation
- Dashboarding
- Business analysis
- Sales performance analysis

## Business Questions

The analysis explores questions such as:

- Which genres generate the highest sales?
- Which regions perform strongest overall?
- Which states and publishers contribute most to sales?
- How do weekly sales trends change across Q1?
- How do sales vary by order size?
- What is the relationship between sales and quantity?

## Key Insights

- History generated the highest sales by genre, followed by Sci-Fi and Non-Fiction.
- The West region showed the strongest regional sales performance.
- Publisher performance varied significantly, with a small group of publishers contributing higher sales.
- Large orders generated substantially higher sales than medium and small orders.
- Running total sales showed the West region leading throughout Q1.
